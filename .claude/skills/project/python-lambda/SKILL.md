# Python サーバレス（プロジェクト用テンプレート）

この Skill は**任意**である。クラウドや実行環境が未確定のときは
質問ドキュメントでプロバイダ・デプロイ方式を確定してから参照すること。

## 適用条件

- Human または質問ドキュメントの回答で、Python による関数実装が選ばれていること
- ターゲットがサーバレス実行（例: 各クラウドの関数サービス）であること

## 作業上の原則

- アカウント ID・リージョン・関数名などの識別子はハードコードしない。プレースホルダまたは設定参照にする
- シークレットは環境変数・シークレットマネージャ経由とし、コードやログに出さない
- プロバイダ固有の SDK やイベント型は、選定されたクラウドに合わせて読み替える（このテンプレートはプロバイダ中立の規約のみ示す）

## 推奨する確認（実装前に Human へ質問可能な項目）

- ランタイムのマイナーバージョンと依存管理方法（例: レイヤー、コンテナイメージ）
- 同期/非同期ハンドラ、タイムアウト、メモリ上限
- テスト方針（ローカルエミュレータの有無、モックの範囲）

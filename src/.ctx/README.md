# `.ctx/` ディレクトリ（LLM 用・ローカル）

このディレクトリは **その親ディレクトリ（例: `src/`）のコンテキスト** を Cursor と Claude Code で共有するための作業メモ用である。

## 何を書くか

- そのディレクトリの責務・モジュール境界の一言説明
- 直近の大きな変更の背景（リファクタ理由など）
- エージェントに読ませたい注意点（命名、依存の向き）

## 何を書かないか

- シークレット・認証情報・本番 URL
- コミットすべき仕様の正本（それは `spec/` や `docs/project/`）

## 更新タイミング（`CLAUDE.md` と同様）

- **新規ファイルやサブディレクトリを追加したとき** → この README を更新
- **大きなリファクタのあと** → 変更背景を追記
- **小さなバグ修正のみ** → 更新不要でよい

## Git

`src/.ctx/` 以外の `.ctx/` も同様に使える。`.gitignore` により **この README 以外** はコミットされない想定（テンプレートとして README のみ追跡）。
